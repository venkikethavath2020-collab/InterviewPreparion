# CSS Interview Quick Revision (30 Questions)

---

### 1. What does CSS stand for?

**Answer:** Cascading Style Sheets
**Explanation:** CSS is used to style and layout HTML elements.

---

### 2. Which CSS property changes text color?

**Answer:** `color`
**Explanation:** Sets the color of text content.

---

### 3. Which CSS property changes background color?

**Answer:** `background-color`
**Explanation:** Applies color to the background of an element.

---

### 4. Which property is used to change font size?

**Answer:** `font-size`
**Explanation:** Controls the size of text.

---

### 5. Default position value in CSS?

**Answer:** `static`
**Explanation:** Elements follow the normal document flow.

---

### 6. Which CSS property makes text bold?

**Answer:** `font-weight`
**Explanation:** Setting `font-weight: bold` increases text thickness.

---

### 7. Difference between class and id selector?

**Answer:** `id` is unique, `class` is reusable
**Explanation:** ID selector `#id` has higher specificity than `.class`.

---

### 8. Which selector has the highest specificity?

**Answer:** Inline styles
**Explanation:** Inline styles override other CSS rules except `!important`.

---

### 9. How to include external CSS?

**Answer:** `<link>` tag
**Explanation:** Used to link an external CSS file to an HTML document.

```html
<link rel="stylesheet" href="styles.css">
```

---

### 10. Which property hides an element but keeps space?

**Answer:** `visibility: hidden`
**Explanation:** Element becomes invisible but still occupies space.

---

### 11. Which property removes an element from layout?

**Answer:** `display: none`
**Explanation:** Element is completely removed from the layout.

---

### 12. Which property controls spacing inside border?

**Answer:** `padding`
**Explanation:** Space between content and border.

---

### 13. Which property controls space outside border?

**Answer:** `margin`
**Explanation:** Space outside the element boundary.

---

### 14. Which CSS unit is relative to parent font size?

**Answer:** `em`
**Explanation:** Scales relative to the parent element’s font size.

---

### 15. Which unit is relative to root element?

**Answer:** `rem`
**Explanation:** Relative to the `<html>` font size.

---

### 16. Which property is used for flex layout?

**Answer:** `display: flex`
**Explanation:** Enables Flexbox layout.

---

### 17. Which property aligns items horizontally in flex?

**Answer:** `justify-content`
**Explanation:** Controls alignment along the main axis.

---

### 18. Which property aligns items vertically in flex?

**Answer:** `align-items`
**Explanation:** Controls alignment along the cross axis.

---

### 19. Which property is used to float elements?

**Answer:** `float`
**Explanation:** Positions elements to the left or right.

---

### 20. Which property clears float?

**Answer:** `clear`
**Explanation:** Prevents floating elements from overlapping.

---

### 21. Which technique is used to make responsive layouts?

**Answer:** Media Queries
**Explanation:** Applies styles based on screen size.

---

### 22. Syntax of media query?

**Answer:**

```css
@media (max-width: 768px) { }
```

**Explanation:** Applies CSS rules when screen width is 768px or smaller.

---

### 23. Which property controls element stacking order?

**Answer:** `z-index`
**Explanation:** Higher values appear on top of lower ones.

---

### 24. Which CSS property adds shadow?

**Answer:** `box-shadow`
**Explanation:** Adds shadow effects around elements.

---

### 25. Which property makes rounded corners?

**Answer:** `border-radius`
**Explanation:** Rounds the corners of elements.

---

### 26. Which property adds animation?

**Answer:** `animation`
**Explanation:** Applies keyframe-based animations.

---

### 27. Which property adds transition effect?

**Answer:** `transition`
**Explanation:** Creates smooth changes between CSS values.

---

### 28. Which property changes mouse cursor?

**Answer:** `cursor`
**Explanation:** Changes pointer appearance.

Example:

```css
cursor: pointer;
```

---

### 29. Which CSS property makes element fixed to viewport?

**Answer:** `position: fixed`
**Explanation:** Element stays fixed even when scrolling.

---

### 30. Which CSS property changes element opacity?

**Answer:** `opacity`
**Explanation:** Controls transparency from `0` to `1`.

---
