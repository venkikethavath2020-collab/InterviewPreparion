# HTML Top 30 MCQs (With Answers + Short Explanations)

---

### 1. What does HTML stand for?

**Answer:** HyperText Markup Language
**Explanation:** HTML structures web content using tags and elements. It is not used for programming logic.

---

### 2. HTML is a

**Answer:** Markup language
**Explanation:** HTML defines page structure and content, not logic like loops or conditions.

---

### 3. Which tag is used to create a hyperlink?

**Answer:** `<a>`
**Explanation:** The `<a>` tag creates links between pages using the `href` attribute.

---

### 4. Which tag is used to insert an image?

**Answer:** `<img>`
**Explanation:** The `<img>` tag embeds images using `src` and `alt` attributes.

---

### 5. Which attribute provides alternative text for images?

**Answer:** `alt`
**Explanation:** The `alt` attribute improves accessibility and displays text if the image fails to load.

---

### 6. Which tag is used for the largest heading?

**Answer:** `<h1>`
**Explanation:** `<h1>` represents the most important heading on a page.

---

### 7. Which tag is used to create a line break?

**Answer:** `<br>`
**Explanation:** `<br>` inserts a line break without creating a new paragraph.

---

### 8. Which tag is used to create an unordered list?

**Answer:** `<ul>`
**Explanation:** `<ul>` creates a bulleted list, typically containing `<li>` items.

---

### 9. Which tag is used to create a table row?

**Answer:** `<tr>`
**Explanation:** `<tr>` defines a row within an HTML table.

---

### 10. Which tag defines a table header cell?

**Answer:** `<th>`
**Explanation:** `<th>` creates bold and centered header cells in a table.

---

### 11. What is the correct HTML tag for paragraph?

**Answer:** `<p>`
**Explanation:** `<p>` defines a paragraph of text.

---

### 12. Which HTML element is block-level?

**Answer:** `<div>`
**Explanation:** `<div>` starts on a new line and takes full available width.

---

### 13. Which HTML element is inline?

**Answer:** `<span>`
**Explanation:** `<span>` occupies only the space needed for its content.

---

### 14. What is the purpose of the `<title>` tag?

**Answer:** Defines page title
**Explanation:** The title appears in the browser tab and helps search engines identify the page.

---

### 15. Which tag is used to create a form?

**Answer:** `<form>`
**Explanation:** `<form>` collects user input and sends data to a server.

---

### 16. Which input type hides the entered text?

**Answer:** `password`
**Explanation:** Password fields mask typed characters for security.

---

### 17. Which attribute groups radio buttons together?

**Answer:** `name`
**Explanation:** Radio buttons with the same `name` behave as a group.

---

### 18. Difference between id and class?

**Answer:** `id` is unique, `class` is reusable
**Explanation:** IDs identify a single element; classes can be used on multiple elements.

---

### 19. Which HTML attribute disables an input field?

**Answer:** `disabled`
**Explanation:** Disabled inputs cannot be edited or submitted with the form.

---

### 20. Which attribute makes an input field non-editable but still submitted?

**Answer:** `readonly`
**Explanation:** Readonly inputs cannot be modified but are submitted with the form.

---

### 21. Which tag is used for navigation links?

**Answer:** `<nav>`
**Explanation:** `<nav>` contains major navigation links of the page.

---

### 22. Which tag represents independent content?

**Answer:** `<article>`
**Explanation:** `<article>` represents standalone content such as blog posts or news articles.

---

### 23. Which tag groups related content?

**Answer:** `<section>`
**Explanation:** `<section>` organizes related content into logical sections.

---

### 24. What is the purpose of `<meta charset="UTF-8">`?

**Answer:** Character encoding
**Explanation:** It ensures special characters display correctly.

---

### 25. Which meta tag is important for responsive design?

**Answer:** `viewport`
**Explanation:** It controls layout scaling on mobile devices.

Example:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

### 26. Which tag is used to emphasize text semantically?

**Answer:** `<em>`
**Explanation:** `<em>` indicates emphasis and adds semantic meaning.

---

### 27. Which tag is used to make text bold semantically?

**Answer:** `<strong>`
**Explanation:** `<strong>` indicates important text and improves accessibility.

---

### 28. HTML comments are written as

**Answer:**

```html
<!-- comment -->
```

**Explanation:** Comments are ignored by the browser and used for documentation.

---

### 29. Which tag is used to play video files?

**Answer:** `<video>`
**Explanation:** `<video>` embeds video content directly in HTML.

---

### 30. What is the purpose of the `<label>` tag?

**Answer:** Improves form accessibility
**Explanation:** `<label>` connects descriptive text with input fields for better usability.

Example:

```html
<label for="name">Name</label>
<input id="name" type="text">
```
